== Description ==

A plugin that allows you to display custom messages on WooCommerce variable products. It allows to configure custom messages on product variations that are displayed when a variation is selected.

We would love to hear your feedback in a [review](https://wordpress.org/support/plugin/woo-variation-message/reviews/). It helps us improve and expand the plugin according to your needs.

If you have any questions or feature requests, feel free to contact us via support@webkinder.ch.

== Installation ==

1. Upload the plugin files to the '/wp-content/plugins/woo-variation-message' directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress.
1. Use the 'WooCommerce->Settings' screen to configure your messages. Goto 'Products->Select Product->Variations->Select Variation' and select the Checkbox.
